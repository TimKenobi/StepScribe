Requirements

System Requirements

StepScribe runs on any modern computer. Local AI is optional but recommended for full privacy.

💻 StepScribe App

  • macOS 11+ (Apple Silicon & Intel), Windows 10+, or Linux (Ubuntu 20.04+)
  • 4 GB RAM minimum
  • 500 MB disk space for the app
  • No internet required (except for cloud AI)

🤖 Local AI (Ollama)

  • Ollama installed (free, open source)
  • 8 GB RAM minimum (16 GB recommended)
  • 5 GB disk space per AI model
  • Recommended: Psychologist by ALIENTELLIGENCE

💡 Don't want to run AI locally? StepScribe also works with cloud providers — Grok (xAI), OpenAI, Anthropic, or any OpenAI-compatible endpoint. Just bring your API key.

💡 Which download do I need?

Mac — Apple Silicon vs Intel: If your Mac was made in 2020 or later, it almost certainly has Apple Silicon (M1, M2, M3, or M4). To check: click the menu → "About This Mac" — if the chip says "Apple M…" choose Apple Silicon; if it says "Intel" choose Intel Mac.

Windows: One installer covers all modern Windows PCs (64-bit).

Linux: Both packages are ARM64 (aarch64). AppImage runs on any distro; .deb is for Debian/Ubuntu-based systems.
